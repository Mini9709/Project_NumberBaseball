숫자 야구 게임 실행 방법
0. 난이도 설정
    - 3~5의 숫자를 입력하여 자릿수 결정
    - 결정 후 바로 게임 시작
1. 게임 시작
    - 설정한 자릿수에 따라 무작위 숫자 생성(기본 자릿수 : 3)
    - 정답일 것 같은 숫자 입력
    - 힌트 출력(숫자와 자리가 맞으면 Strike, 숫자만 맞으면 ball, 입력한 숫자가 모두 틀리면 out)
    - 정답 입력 시 닉네임 입력
    - 메뉴로 이동
2. 게임 기록
    - 저장된 순서대로 기록 출력
    - 1. 순위를 3위까지 출력
    - 2. 기록되어 있는 데이터 모두 삭제
    - 3. 메뉴로 이동
3. 종료
